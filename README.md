# Competitive Analysis of Cache Replacement Algorithms

## Overview

This research project evaluates the performance of three fundamental cache replacement algorithms—Least Recently Used (LRU), Least Frequently Used (LFU), and Belady's Optimal Algorithm—across multiple synthetic memory access workloads.

The project investigates how different workload characteristics influence cache efficiency by comparing cache hits, misses, evictions, and hit rates. Results demonstrate that no single caching policy performs best in every scenario and that workload behavior is the primary factor when selecting an effective cache replacement strategy.
